### 创建新版本库

```
git clone git@code.aliyun.com:gobtx/btx-frontend.git
cd btx-frontend
touch README.md
git add README.md
git commit -m "add README"
git push -u origin master

```

---


### 已存在的文件夹或 Git 仓库


```
cd existing_folder
git init
git remote add origin git@code.aliyun.com:gobtx/btx-frontend.git
git add .
git commit
git push -u origin master
```
